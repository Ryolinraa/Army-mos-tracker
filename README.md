# Army-mos-tracker
Check previous versions of an MOS

Army MOS Lineage & Consolidation Tracker
Bridging the Gap Between Historical JST Records and Modern Designations (1985–Present)

The Problem
Military Occupational Specialty (MOS) codes are not static. Over the decades, the Army has undergone massive structural changes—most notably the 2004 conversion from numerical to alphanumeric codes and the frequent merging of specialized roles into broader "parent" MOSs. For veterans looking at a Joint Service Transcript (JST), a code held in 1992 might look completely different today, making it difficult to translate skills to civilian employers or academic advisors.

The Solution
This app acts as a "Time Machine" for Army MOSs. By entering a specific code and the year it was held, the user can:

Trace the Lineage: See exactly what that MOS turned into (e.g., how 63B became 91B).

Identify Mergers: Discover if a specific job was consolidated into a larger field (e.g., 11M merging into 11B).

Verify History: View a chronological timeline of designations from 1985 to the Present.

Key Features
Enlisted & Warrant Officer Support: Comprehensive tracking for both cohorts.

Consolidation Logic: Specifically designed to show when multiple historical jobs became one modern job.

Mobile-Friendly: Built to be used on the go during recruitment or counseling sessions.
